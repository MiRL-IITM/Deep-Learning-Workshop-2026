# CERTIFICATE COURSE IN ARTIFICIAL INTELLIGENCE

## Deep Learning Workshop

Welcome to the Deep Learning Workshop repository! This workshop provides a hands-on introduction to deep learning concepts and their implementation using PyTorch, offered as part of the Certificate Course in Artificial Intelligence.

## Workshop Structure

The workshop is designed as a series of sessions, starting with a PyTorch introduction and progressing through advanced topics like CNNs and RNNs. Every session contains demonstration notebooks followed by Exercises (named as `E*.ipynb`).

- **Pre-requisites**: Basic Python programming and environment setup (Google Colab / Local).
- **S0 - PyTorch Introduction and Linear Regression**: Fundamentals of PyTorch, tensors, and basic workflow.
- **S1 - Convolutional Neural Networks (CNN)**: Image classification and CNN foundations.
- **S2 - RNN Sequence Modelling**: Introduction to Recurrent Neural Networks and sequence data.
- **S3 - RNN Sentiment Analysis**: Applying RNNs to natural language processing and classification.
- **S4 - RNN Encoder Decoder Models**: Advanced sequence-to-sequence models using encoder-decoder architectures.
